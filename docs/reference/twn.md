# Een package voor de TWN-lijst

De bedoeling van 'twn' is om de TWN-lijst beschikbaar te maken in R en
om er makkelijk mee te kunnen werken. De package biedt diverse functies
die het eenvoudig maken om informatie van een taxon op te vragen.

Zie ook:

- [twn_lijst](https://redtent.github.io/twn/reference/twn_lijst.md) en
  [twn_statuscodes](https://redtent.github.io/twn/reference/twn_statuscodes.md)

- [twn_info](https://redtent.github.io/twn/reference/twn_info.md)

- [`match_parent()`](https://redtent.github.io/twn/reference/match_parent.md)

Voor meer informatie zie de package website:
<https://redtent.github.io/twn/>
