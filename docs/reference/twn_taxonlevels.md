# TWN taxonlevels

De TWN-lijst heeft aan ieder taxon een taxonlevel toegekend.
`twn_taxonlevels` is een geordende factor met alle beschikbare
taxonlevels.

## Gebruik

``` r
twn_taxonlevels
```

## Formaat

Geordende factor

## Voorbeelden

``` r
twn_taxonlevels
#>  [1] Subforma      Forma         Varietas      Subspecies    Cultivar     
#>  [6] Species       Nothospecies  Species combi Sectio        Subgenus     
#> [11] Genus         Nothogenus    Genus combi   Tribus        Subfamilia   
#> [16] Familia       Superfamilia  Infraordo     Subordo       Ordo         
#> [21] Superordo     Infraclassis  Subclassis    Classis       Superclassis 
#> [26] Infraphylum   Subphylum     Divisio       Phylum        Superphylum  
#> [31] Infraregnum   Subregnum     Regnum        Superregnum   Imperium     
#> [36] Superimperium
#> 36 Levels: Subforma < Forma < Varietas < Subspecies < Cultivar < ... < Superimperium
```
